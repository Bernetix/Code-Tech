Jak wygląda składnia CSS?

p{ color: pink; font-size: x-large; background-color: darkblue; }

Wyjaśnienie:

  • p jest selektorem, odpowiednio użytym w pliku html
  • font-size, color oraz background-color odpowiadają za cechy danego bloku / tekstu
  • pink, x-large oraz darkblueto wartości argumentów to dzięki nim zmieniamy kolor lub rozmiar
  • Po użyciu każdego argumentu powinien znajdować się średnik ;
  • Argumenty selektora zawsze powinny być ujęte w klamry otwierające { oraz zamykające }
