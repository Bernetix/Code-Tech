HTML
HTML jest jednym z języków znacznikowych stosowanym do tworzenia dokumentów hipertekstowych oraz stron interentowych.
Rozszerzeniem pliku omawianego przez nas języka jest - .html
Przykładowy plik HTML:
Pobierz mnie!
Podstawowe znaczniki w języku HTML:

Akapit

<p> To jest akapit </p>

Nagłówek

<h1> To jest nagłówek </h1>

Dostępnych jest 6 wielkości nagłówków: h1, h2, h3, h4, h5 oraz h6

Znacznik obrazka

<img src="obrazek.jpg" width="150px" height="100px" alt="obrazek">

Znacznik img moze mieć wiele argumentów. Są to m.in. alt (tekst alternatywny, pojawiający się gdy obraz jest niedostępny z róznych powodów), height i width (wysokość oraz szerokość) oraz oczywiście src (czyli argument w którym podajemy źródło z którego obraz ma się wyświetlić: link z internetu lub ścieka z dysku)

Hiperłącza

<a href="index.html target=_blank>Strona główna</a>

Znacznik hiperłącza może prowadzić zarówno do podstron, jak i umożliwa utworzenie pobieralnego pliku. Wszystko zależy od ścieżki którą wpiszemy w polu "href". Przydatne są także argumenty "target". Mogą przybierać one wiele form, jednak najpopularniejsze z nich to _self (otwieranie odnośnika w tej samej karcie) oraz _blank (dzięki niemu otworzymi odnośnik w nowej karcie)

Przycisk

<button>Przycisk</button>

Przycisk używany jest w formularzach lub innych interakcjach na stronie. Używając argumentów możemy z powodzeniem używać go do wykonywania skryptów napisanych najczęścniej w Java Script lub PHP. Użycie przycisku w Java Script omówione zostanie w kursie z JS.
